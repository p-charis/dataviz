Big Tech's Funding of Journalism

Info

This is a project developed by Charis Papaevangelou in collaboration with The Centre for Media, Technology & Democracy of McGill Univeristy meant to track Big Tech's funding on journalism.
Data principally concern the Facebook Journalism Project (now Meta Journalism Project), that began in 2018 as Facebook's $300 million global investment on news media organizations.
Data were primarily taken from the FJP's blog (blog entries, press releases, announcements, etc.) and were gathered using basic HTML scraping and a lot of manual work. Some were also found in FJP's partners' sites.
Data are up-to-date until November 9, 2021.


There are 3 datasets: beneficiaries, intermediaries and FJP programs. You can filter data by selecting specific Regions, Countries or even Funding Program. You can also search for a specific news media organization. As this is an on-going project, more granular data will be added to each beneficiary.

Facebook Journalism Project - Funding Intermediaries

We define 'funding intermediaries' as third-parties (both non and for-profit organizations) that facilitate the allocation of Facebook's grants directly (e.g., by distributing Facebook's funds) or indirectly (e.g., by reviewing grant applications for Facebook)
